# easypay

独立支付集成平台，集成微信支付、支付宝支付


## 项目初衷

在日常工作中，资金的收付是很重要的一环，但每次都需要重新开始集成到项目中，导致很麻烦。所以结合项目经验、自己在用友的部分经验，自己摸索整理一套支付中台系统，用于对接当前的支付逻辑。

为了更灵活，考虑采用插件的逻辑进行开发，对外提供api或sdk方便对接到其它系统中。

## 核心功能


### 统计分析

- 收支统计
- 费用统计


### 微信支付

对接微信商户支付

- 参数设置
- 公众号支付
- 小程序支付
- app支付
- 对账
- 转账到微信零钱
- 转账到银行卡


### 支付宝支付

- H5支付
- app支付
- 对账
- 转账到支付宝账户
